Cat Log Puzzle (キャット・ログ・パズル)
〜 数学の美しさと遊び心が溶け合う、知的な迷宮へようこそ 〜

🧩 概要 / Overview

Cat Log Puzzle は、提示された3つの数字と数学記号を駆使して、答えを「10」にする計算パズルアプリです。
単なる計算ゲームではなく、階乗（!）、対数（log）、組み合わせ（nCr）といった高度な数学概念を用いることで、より「美しく、エレガントな解法」を追求することを目的としています。

Cat Log Puzzle is a mathematical puzzle app where you use three given numbers and various operators to reach the target number "10".
Beyond a simple arithmetic game, it encourages players to find "elegant solutions" using advanced concepts like factorials, logarithms, and combinations.

🐱 キャラクター / Character

Cat Log (キャット・ログ)
不思議の国から迷い込んだ、数学を愛するチェシャ猫。ニヒルな笑みを浮かべながら、あなたが数式の迷宮で立ち止まった時にだけ、謎めいたヒントを囁きます。
"A mystical Cheshire cat from Wonderland. He whispers cryptic hints only when you're lost in the labyrinth of equations."
